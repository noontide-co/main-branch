## Format

```markdown
---
date: 2026-01-13
topic: Topic Name
sources: [url1, url2]
tags: [tag1, tag2]
---

# Topic Name

## Summary
[2-3 sentences]

## Key Points
- Point 1
- Point 2

## Sources
- [Title](url) - key insight
```

## Naming

`YYYY-MM-DD-topic-slug.md`

Example: `2026-01-13-meta-andromeda-algorithm.md`
